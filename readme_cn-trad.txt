1. WinFlip 是什麼?
 
    WinFlip 使 Windows XP中的視窗擁有翻轉的功能。(類似Windows Vista的 Flip 3D)
________________________________________________________________________________

2. 系統需求

    作業系統      : Windows XP
    額外需求      : DirectX9.0c(在 SP1 或更早的系統上)
                    在微軟的網站有提供下載
                    http://www.microsoft.com/directx/
________________________________________________________________________________

3. 如何使用
   
    1:執行 Winflip.exe，Winflip的圖示會在工作列右方出現。
  
    2:當 Winflip 啟動後，用以下按鍵來翻轉在桌面上的視窗。

      Win鍵 + Tab           (向前翻轉)
      Win鍵 + Shift + Tab   (向後翻轉)

    3:要變更顯示品質，在圖示按下右鍵，選擇清單中的"品質設定"，
      再從次選單中選擇所需的品質。

    4:要變更視窗大小(當翻轉視窗時)，在圖示按下右鍵，選擇清單中
      的"視窗大小設定"，再從次選單中選擇所需的大小。

    5:要變更反鋸齒品質，在圖示按下右鍵，選擇清單中的"反鋸齒設定"，
      再從次選單中選擇0至16。程式將自動重新啟動。

    附註：電腦的硬體設備決定反鋸齒品質。Winflip 在程式運作前
          會檢查硬體支援度，使得能選擇的數字皆能被硬體支援。

    6:如果翻轉十分緩慢，你可以選擇以省略畫面的模式執行。
      要選擇時，在圖示按下右鍵，選擇清單中的"當速度過慢時省略畫面"。

    7:要變更快捷鍵設定，在圖示按下右鍵，選擇清單中的"選項"，
      並根據對話框的指示完成設定。

    8: 以下的滑鼠操作已被支援
      a.要開始翻轉視窗，你可以選擇以下的操作方法。
        (i)  滑鼠手勢
             用滑鼠指標在任意處畫一個圓。
	     [可以在選項中設定順時鐘的圓 或/且 逆時鐘的圓 是否可被接受]
        (ii) 在桌面邊緣處點擊。
        (iii)點擊滑鼠中鍵。
      b.翻轉視窗時，可以使用滑鼠的滾輪。
      c.選擇視窗時，只需點選你想選擇的視窗。

    9:要離開時，在圖示按下右鍵，選擇"結束"。

   10: 在0.50版中新增了"例外檔案設定"的功能，允許使用者設定哪些視窗不要在翻轉              
       清單中顯示。
       若要建立例外清單，在圖示按下右鍵並選擇"選項"。 在左側的標籤選擇"其它"。 
       在"目前正在執行的視窗"選擇不想出現在翻轉清單中的程式。可以使用Ctrl 鍵  
       選擇多個檔案。
       接著，點擊"新增至清單"按鈕。
       剛剛所選擇的視窗應該會出現在"例外清單"中。
       若要從例外清單中移除，在例外清單中選擇想解除的程式後，
       按下"從清單中移除"。 

   11: TrueTransparency(v.0.9.2) 已從版本 WinFlip 0.50 開始支援
       不需要額外設定即可使用。
       WinFlip 會自動偵測TrueTransparency 然後將標題列格式轉變
       TrueTransparency 所指定的標題列格式。

   12: 開機時自動執行(從版本 WinFlip 0.50開始支援)
       要在開機時自動執行WinFlip，在圖示按下右鍵並選擇 "選項"。在左側的標籤
       選擇 '其它' 並按下 '開機時自動執行'。若要從開機自動執行列移除，請按下"開   
       機時不要自動執行"。

   13: 從捷徑圖示中啟動翻轉 (從版本 WinFlip 0.50開始支援)
       在桌面，快速啟動列，或任何地方放置捷徑。在WinFlip執行時用滑鼠點擊兩下，則 
       啟動翻轉。
       你也可以選擇從快速啟動列的WinFlip達到相同的效果，只需用左鍵點擊該圖示即可。
