1. What is WinFlip?

    WinFlip emulates Windows Vista's "Flip 3D" effect for Windows XP. Using the
    hotkey of your choice, you can find the window you're looking for in a fast,
    clean 3D interface.
________________________________________________________________________________

2. Requirements

    OS      : Windows XP
    Library : DirectX9.0c (2004 Summer) (only if SP1 or earlier)
              available at Microsoft Corporation under:
              http://www.microsoft.com/directx/
________________________________________________________________________________

3. How to use

    1: Double-click on WinFlip.exe. WinFlip should appear in the tasktray.

    2: While WinFlip is running, flip windows on the desktop using the following
    keyboard shortcuts:

        Win + Tab          (Forward)
        Win + Shift + Tab  (Backward)
        
        You can decide whether to use the Alt or Windows key by right-clicking
        on the tray icon and selecting 'Options'.

    3: To adjust the texture quality, right click the icon in the tasktray, and
    select "Texture quality" in the menu. Then choose a texture quality setting
    from the submenu.

    4: To change the window size (when flipping), right click the tasktray icon,
    and select "Display size" in the menu. Choose a size from the submenu.

    5: To change the anti-aliasing quality, right click the icon in the 
    tasktray, and select "Anti-aliasing" in the menu. Then choose 0 to 16 from
    the submenu. The application restarts automatically.

        note: The anti-aliasing quality depends on your computer's hardware.
        WinFlip checks the hardware support when starting up the program, and
        enables the menu items which are supported by the hardware.
        Lower anti-aliasing may result in windows with jagged edges whilst
        flipping, but should make WinFlip faster.

    6: If the flip function is very slow, you can choose the 'drop frames' mode.
    Right click the icon in the tasktray, and select "Drop frames when slow" in
    the menu. 

    7: To change the keyboard shortcuts, right click the icon in the tasktray,
    and select "Options" in the menu. The options box should open. 
    Follow the instructions in the dialogbox to complete the shortcut settings.

    8: The following mouse operations are supported:
      a. To begin flipping, you can choose following operations...
        (i)  Mouse gesture:
             Draw a circle anywhere on the desktop using the mouse cursor.
             You can enable drawing clockwise and/or counter-clockwise.
        (ii) Click edge of desktop
        (iii)Click the middle button
      b. To flip between windows, you can use the mouse wheel.
      c. To select a window, just click the window you want to choose.
      d. To bring a window to the front click middle button on the
        target window.

    9: To exit WinFlip, right click the icon in the tasktray, and select
       "Quit" in the menu.

   10: An exclusion list was introduced in version 0.50, and allows you to
   choose which windows not to show whilst flipping.
       To set up the exclusion list, right click the icon in the tasktray and
       select "Options" in the menu. The options screen should appear. Select
       the "Other" tab.
       Select the window(s) which you do not want to show whilst flipping in the
       'Current windows' list. Use the Ctrl key to select more than one.
       After that, press the 'Add to list' button.
       Windows which you selected should appear in the 'Exclusion list'.
       To stop hiding excluded windows, select the line(s) which you want to
       remove from the list and press 'Remove from list'.

   11: TrueTransparency(v.0.9.2) is supported from version 0.50.
       No setting is required to enable this effect.
       WinFlip detects the TrueTransparency automatically and draws the tile
       like TrueTransparency does.

   12: Add shortcut to startup (supported from version 0.50)
       To run WinFlip at startup, right click the icon in the tasktray and
       select 'Options' in the menu. The options screen should appear. Select
       the 'Other' tab, and press 'Run automatically'. To remove the shortcut
       from startup, press the 'Don't run automatically' button.

   13: Start flip from the shortcut icon (supported from version 0.50)
       Put a shortcut to WinFlip on the desktop, quick launch or anywhere else
       you want. Double click it while winflip is running, then flip starts.
       You can also left-click on the icon in the tasktray to start flipping.
